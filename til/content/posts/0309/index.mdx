## 배열

```js
let myNumber = [79, 98, 86, 61, 96];
```

<br />

배열 기초
`index`, `element`, `length`,
`arr[0]`

```js
// myNumber 배열 안 3번째 index의 element
myNumber[3]; // 61

// 배열의 길이를 알아낼 수 있다
myNumber.length; // 5
```

<br />

배열에서 특정 인덱스(index)의 요소(element)를 조회 및 변경

```js
myNumber[0] = 100;
// [100, 98, 86, 61, 96]
```

<br />

이중 배열

```js
let myNumber = [
    [13, 30],
    [73, 8],
    [44, 17],
];
// myNumber의 1번째 index의 값은?
// myNumber의 1번째 index값의 0번째 인덱스 값은?
myNumber[1]; // [73, 8]
myNumber[1][0]; // [73]
```

<br />

for 문과 배열을 이용

```js
for (let n = 0; n < myNumber.length - 1; n++) {
    console.log(myNumber[n]);
}
```

<br />

응용 for문

```js
let myNum = [10, 20, 40, 10];
let sum = 0;
for (let i = 0; i < myNum.length; i++) {
    sum += myNum[i];
}
// 80
```

<br />

띄어쓰기 (" ") 로 문자열을 구분하여, 배열로 바꾸기 `str.split(" ")` `.join(" ")`

```js
let str = "ciao ciao the cat";
let arr = a.split(" "); // ["ciao", "ciao", "the", "cat"]
arr.join(" "); // "ciao ciao the cat"
```

<br />

배열 요소(element)의 조회, 추가, 삭제, 복사, 분리
`unshift`, `shift`, `slice`, `length`

```js
// 앞에 있는 요소 삭제
let arr = ["code", "states"];
arr.shift(); // "code"

// 배열 앞에 요소 추가
arr.unshift("creative"); // ["creative", "states"]
```

<br />

배열의 요소(element)를 추가 및 삭제 `push`, `pop`

```js
// 배열 끝에 값을 추가
myNumber.push(96);
console.log(myNumber); // [79, 98, 86, 61, 96, 96]

// 배열 마지막 값을 삭제
myNumber.pop(); // 96
console.log(myNumber); // [79, 98, 86, 61, 96]
```

<br />

자바스크립트의 특정 값이 배열인지 아닌지 판별 `Array.isArray`

```js
Array.isArray(myNumber); // true
Array.isArray("문자열"); // false
```

<br />

자바스크립트의 특정 값이 배열에 포함되어 있는지 확인 `indexOf`, `includes`

```js
let words = ["Radagast", "the", "Brown"];
words.indexOf("the") // 1 (index 의 값이 반환)
words.indexOf("Brown") // 2
words.indexOf("없는 단어") // -1
// 특정한 값이 배열에 포함되어 있지 않으면 -1이 반환된다.
// indexOf는 대문자와 소문자 값을 구분한다

// #boolean 값으로 반환하기
words.indexOf("없는 단어") !== =1 //false
words.includes("Radagast") // true
```

## 참조형 데이터 타입

    참고 - 인사이드 자바스크립트

<br />

```js
const a = [1, 2, 3];
let b;
// b 에 a 할당
b = a;
b.push(4);
console.log(b); // [1, 2, 3, 4]
console.log(a); // [1, 2, 3, 4]
```

b 에 a를 할당하게 되면 [1, 2, 3]이 들어가는게 아니라 [1, 2, 3]을 갖고있는 메모리의 주소를 할당받게 된다.

왜일까!

_(아래 자세히 설명)_

<br />

### 자바스크립트의 값들은 크게 기본 타입과, 참조 타입으로 나뉜다.

-   기본타입
    -   숫자(Number)
    -   문자열(String)
    -   불린값(Boolean)
    -   undefined
    -   null
-   참조 타입
    -   객체(Object)
        -   배열(Array)
        -   함수(function)
        -   정규 표현식

자바스크립트에서는 기본 타입을 제외한 모든 값이 객체이다. 배열, 함수 모두 자바스크립트 객체로 표현된다.

객체란 `이름(key):값(value)`형태의 프로퍼티들을 저장하는 컨테이너이다.

기본 타입은 하나의 값만 가지는 데 비해, 참조타입인 객체는 여러 개의 프로퍼티들을 포함할 수 있다. 이러한 객체의 프로퍼티는 기본 타입의 값을 포함하거나, 다른 객체를 가리킬 수도 있다.

이러한 프로퍼티의 성질에 따라 객체의 프로퍼티는 함수로 포함할 수 있다. 이러한 프로퍼티를 자바스크립트에서는 메서드라고 부른다.

```js
let a = 100;
let b = 100;

let objA = {
    value: 100,
};
let objB = {
    value: 100,
};
let objC = objB;

console.log(a === b); //true
console.log(objA === objB); // false 1️⃣
console.log(objB === objC); // true 2️⃣
```

변수 a와 b는 숫자 100을 저장하고 있는 **기본 타입**의 변수다. 기본 타입의 경우 비교 할 때 **값을 비교한다**
1️⃣: 위 코드처럼 objA와 objB는 다른 객체지만, 같은 형태의 프로퍼티값을 가지고 있다. 하지만 두 객체를 비교하면 결과는 1️⃣과 같이 false가 나온다. 그 이유는 기본타입의 경우는 값 자체를 비교해서 일치 여부를 판단하지만, 객체와 같은 참조 타입의 경구 참조값이 같아야 true가 된다.

2️⃣: objB와 objC는 같은 객체를 참조하므로 2️⃣와 같이 true가 나오는 것이다.

## 주의

### 1

자바스크립트에서는 한 번 정의된 문자열은 변하지 않는다.

```js
let str = "text";
str[0] = "T";
console.log(str); // test

let arr = [1, 2, 3];
arr[0] = 0;
console.log(arr); // [0, 2, 3]
```

문자열은 문자 배열처럼 인덱스를 이용해서 접근할 수 있지만, 수정은 불가능하다.

<br />

### 2

자바스크립트 내에서 기본적으로 값이 할당되지 않은 변수는 `undefined` 타입이며, `undefined` 타입의 변수는 변수 자체의 값 또한 `undefined` 이다.

** `undefined`는 타입이자, 값을 나타낸다**

```js
let a;
console.log(typeof a); // undefined

let b = null;
console.log(typeof b); // object
```

<br />

### 3

배열 `[] === []`의 값은 `false`이다.
자바스크립트 상에서는 두 배열을 "주소가 다른 두 개의 빈 배열"이라고 생각하기 때문.

## 객체

객체 기초

```js
let user = {
    firstName: "Solhee",
    lastName: "Kim",
    email: "09merus@gmail.com",
    city: "cheonan",
};

// dot notation과 bracket notation
user.firstName; // "Solhee"
user["city"]; // "cheonan"
```

<br />

객체 속성(property)의 추가, 삭제

```js
// 프로퍼티 추가
user.isAdult = true;
// 프로퍼티 삭제
delete user.email;
```

<br />

in 연산자를 이용해 특정 키가 있는지 확인

```js
"lastName" in user; //true
"abcdef" in user; // false
```

<br />

객체를 위한 for문 for ... in 문

```js
for (let key in user) {
    console.log(key);
}
// firstName
// lastName
// email
// city
```
