## CSS Selector

안 써본 선택자만 정리

-   `header > p { }` : 자식 셀렉터
    -   후손 셀렉터와, 자식 셀렉터는 차이가 있다.
-   `section + p { }` : 인접 형제 셀렉터
-   `section ~ p { }` : 형제 셀렉터
-   `a[href] { }` : 속성 셀렉터
-   `p:nth-child(2n+1) { }` : 홀수(2n+1) 짝수(2n+2)
-   `p:nth-last-child() { }` : 마지막에서 부터 순서 계산
-   `p:first-of-type { }` : 형제 요소 중 자신과 일치하는 첫 번째 요소
-   `ul:nth-of-type(2) { }` : 형제 요소 중 자신과 일치하는 n 번째 요소
