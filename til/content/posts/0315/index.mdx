### flex 단축 속성
- `flex: <grow> <shrink> <basis>` (기본값 0 1 auto)
- `flex-flow: <direction> <wrap>`(기본값 row nowrap)

```css
.target {
    flex-grow: 0; 
    flex-shrink: 1;
    flex-basis: auto; 
}
```
- grow와 shrink는 단위가 없다 (비례하는 값)
- grow가 0일 때, basis 크기를 지정하면 그 크기는 보장된다
